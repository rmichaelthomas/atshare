# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),

## [Unreleased]

## [1.5.0] - 2026-03-XX

### Added
- Universal destination support via community JSON registry
- Clipboard fallback for unsupported networks
- Community-driven destination contributions via PR

## [1.0.0] - 2026-03-XX

### Added
- Share Selector web component (`<atshare-selector>`)
- PDS preference record (`social.atshare.preference`)
- AT Protocol OAuth integration
- Bluesky and Mastodon share intent support
- Cross-origin iframe proxy architecture
- Themeable via CSS custom properties
